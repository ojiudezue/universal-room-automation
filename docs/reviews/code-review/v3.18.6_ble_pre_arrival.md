### R1-1 Fix Detail

**Bug:** In `sensor.py` line 7227, `getattr(hvac, '_last_pre_arrival_time', None)` returns a raw `datetime` object to HA state attributes. While HA's state machine can sometimes serialize datetime objects, this is not guaranteed across all serialization paths (history, recorder, REST API) and can cause `TypeError: Object of type datetime is not JSON serializable`.

**Fix:** Apply `.isoformat()` conversion matching the pattern used in the HVAC coordinator's own attributes:
```python
# Before
"last_trigger_time": getattr(hvac, '_last_pre_arrival_time', None),
# After
"last_trigger_time": t.isoformat() if (t := getattr(hvac, '_last_pre_arrival_time', None)) else None,
```
